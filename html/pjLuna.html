Application Design
Oktober 2020-Februar 2021
Caroline Bunk, Pauline Koch, Sophie Vollmer
Redesign einer mobilen Applikation Wie kann eine App über eine einfache Benutzung die vielen körperbezogenen Daten aufnehmen und übersichtlich abbilden, sodass allen Altersgruppen eine optimale Bedienung versprochen werden kann?
Auf dem übersichtlichen Homescreen sind alle wichtigen Informationen auf einen Blick zu finden.
In Sekundenschnelle können die täglichen Eingaben mit wenigen Schritten erledigt und so alle Daten rund um den Körper in einer App gesammelt werden.
Mithilfe der übersichtlichen Statistiken können Körperwerte im Blick behalten werden.
Stetig werden neue Artikel über interessante Themen zum eigenen Körper zur Verfügung gestellt.
Durch Eingabe der persönlichen Daten, kann die volle Individualisierung der App erlebt werden. So wird nur vorgeschlagen, was wirklich interessiert.
